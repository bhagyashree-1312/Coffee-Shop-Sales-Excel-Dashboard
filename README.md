# Coffee-Shop-Sales-Excel-Dashboard
This project focuses on analyzing coffee shop sales data using Microsoft Excel to derive meaningful business insights. The dashboard helps quickly identify top-selling products, revenue-driving categories, and overall sales performance, supporting data-driven decision-making. 

## Author - Bhagyashree Pise

## Project Overview
This project analyzes coffee shop sales data using Microsoft Excel.  
An interactive dashboard was created to track sales performance, customer behavior, and product trends.

## Dataset
The dataset contains transactional sales data, including:
- Date & time of sale
- Product category and product name
- Quantity sold
- Sales amount

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Formatting
- Dashboard Design

## Key Insights
- Identified top-selling products and categories
- Analyzed sales trends over time
- Compared revenue contribution by product type
- Tracked quantity sold vs revenue performance
